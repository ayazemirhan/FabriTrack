# 🚀 FabriTrack – Machine Tracking & Maintenance Management System

FabriTrack is a desktop application developed with C# that enables efficient tracking of industrial machines and management of maintenance processes.

---

## 📌 Overview

This system is designed to manage machine operations, maintenance workflows, and team responsibilities within an organization.

It includes a **role-based structure** with three different user types:

* **Admin**
* **Supervisor (Amir)**
* **Personnel**

---

## 👥 User Roles & Features

### 🔹 Admin

* Manages all machines in the system
* Assigns supervisors to machines
* Has full control over the system

### 🔹 Supervisor (Amir)

* Manages a team of personnel
* Assigns personnel to machines
* Tracks maintenance processes
* Oversees machine status and workflow

### 🔹 Personnel

* Views assigned machines
* Follows maintenance tasks
* Participates in maintenance operations

---

## ⚙️ Features

* Machine tracking system
* Maintenance management
* Role-based authorization
* Team assignment system
* Performance and workflow tracking

---

## 🛠️ Technologies Used

* C#
* .NET (Windows Forms)
* SQL Server

---

## 🎯 Purpose

The main goal of this project is to provide a structured and efficient way to manage machine maintenance processes and improve operational workflow in industrial environments.
